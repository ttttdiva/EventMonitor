# ask

質問専用モード。リポジトリを読んで答えるだけで、変更はしない。

## ルール

- ファイルを編集しない。
- ブランチを作成・切り替えしない。
- commit / push / merge しない。
- 変更が必要だと分かった場合は、分析結果を出して `/work` が必要だと伝える。

## 手順

1. `memory-bank/` を読む。
2. 質問に関係する現在の状態を確認する。
3. 必要なファイルだけ読む。詳細仕様が必要な場合だけ `docs/` を参照する。
4. 必要なら read-only の git 情報を見る。
5. 根拠付きで答える。
