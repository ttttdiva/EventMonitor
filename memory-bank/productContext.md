# productContext.md

## 何を解決するか
- 同人イベント参加・頒布情報の投稿を見逃す問題を解消
- 画像/動画の取りこぼしやバックアップ漏れを防止
- 大量アカウント（949件）の効率的な一括監視

## 想定ユーザー
- 特定アーティスト/サークルのイベント情報を追う運用者
- Discord通知で即時把握したい運用者
- Hydrusで画像整理したい運用者

## 期待される体験
- 監視対象をCSVで管理し、ほぼ自動で運用できる
- Discordチャンネルからアカウント追加が可能（discord_ingest機能）
- イベント関連ツイートが即時Discordに通知される
- メディアはローカル保持され、Hydrusに自動インポートされる
- ダッシュボードで稼働状態をリアルタイム確認可能

## 主要ユースケース
- 監視アカウントの新着チェックとイベント検出
- ログ専用アカウントの全量保存・HFバックアップ・ローカル削除
- Hydrusへのメディア自動インポート（タグ付き）
- Discordダッシュボードによる稼働監視
- AoiTalk連携によるステータスPush
